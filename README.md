## npm 方式安装使用

```js
// 先安装
npm i @wocwin/t-ui-plus
// 在main.js中按下引入
import TuiPlus from '@wocwin/t-ui-plus'
Vue.use(TuiPlus)
```

## 安装依赖 **如果总报错就用 cnpm install**

```shell
npm install

```

## 本地运行 src 中组件示例

```shell
npm run serve
```

## 本地运行 vuepress 中组件文档

```shell
npm run docs:dev

```
