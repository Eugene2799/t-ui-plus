## npm 方式安装使用

```js
// 先安装
npm i @wocwin/t-ui-plus
// 在main.js中按下引入
import TuiPlus from '@wocwin/t-ui-plus'
import '@wocwin/t-ui-plus/lib/style.css'
Vue.use(TuiPlus)
```

## 安装依赖

```shell
pnpm install

```

## 本地运行 examples 中组件示例

```shell
// examples项目（常规的后台管理系统）基于vue3+ts+vite项目
npm run serve
```

## 本地运行 vuepress 中组件文档

```shell
// docs项目(文档demo示例)基于vue3+vite项目
pnpm docs:dev

```
